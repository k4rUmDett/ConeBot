Detailed Description

A class to interface with the Adafruit BNO055 IMU sensor.

The IMU class provides methods to initialize, calibrate, and update the sensor's state, and to retrieve its tilt angle and angular velocity. It supports calibration status checks and prints the calibration data for debugging purposes.
